# Магазин электроники

Проект представляет собой систему управления товарами и категориями для магазина электроники, реализованную на Python с соблюдением принципов ООП.

## Функциональность

### Реализованные классы
- **Класс Product** - представление товара с приватными атрибутами
- **Класс Category** - представление категории товаров

### Основные возможности
- Приватные атрибуты с геттерами и сеттерами
- Автоматический подсчет количества категорий и товаров
- Загрузка данных из JSON файла
- Класс-методы для создания объектов
- Валидация данных (цена, количество)

### Соответствие критериям
- Приватный список товаров в классе `Category`
- Метод `add_product()` для добавления товаров
- Геттер для списка товаров в формате строки
- Класс-метод `new_product()` для создания объектов из словаря
- Приватный атрибут цены с геттером и сеттером
- Валидация цены (только положительные значения)
- Автоматический подсчет количества товаров и категорий
- Покрытие тестами более 75%

## Технологии

- Python 3.8+
- Poetry для управления зависимостями
- Pytest для тестирования
- Pytest-cov для проверки покрытия тестами
- Flake8 для линтинга кода
- Black для форматирования кода
- Git для контроля версий
